# Vishal Bhanopiya – Developer Portfolio

A modern and interactive **developer portfolio website** built with **React, Vite, TailwindCSS, and Three.js**.
This portfolio showcases my **projects, technical skills, GitHub activity, professional experience, and research publication** through an animated and responsive web interface.

The goal of this project is to present my work in a **visually engaging and professional format** for recruiters, collaborators, and developers.

# ⚙ Installation

Clone the repository

```id="install_clone"
git clone https://github.com/VishalBHANOPIYA/Vishal_Portfolio.git
```

Move into project folder

```id="install_cd"
cd vishal-portfolio
```

Install dependencies

```id="install_dependencies"
npm install
```

Run development server

```id="run_dev"
npm run dev
```

---

# 🏗 Build Project

```id="build_project"
npm run build
```

---

# 🚀 Deployment

This portfolio is deployed using **Vercel**.

Steps:

1. Push project to GitHub
2. Import repository in Vercel
3. Deploy project
4. Vercel automatically builds and hosts the site
